# 🧬 +lifeBOSPro (نسخه ۷.۰) – آرشیو شده

> ⚠️ این مخزن **دیگر به‌روز نمی‌شود** و فقط برای اهداف آرشیوی نگهداری می‌شود.

این مخزن شامل نسخه‌ی اولیه (v7.0) از اپلیکیشن **+lifeBOSPro** است. این نسخه یک نمونهٔ اولیه (MVP) با قابلیت‌های پایه مانند داشبورد، برنامه‌ریزی روزانه، پیگیری عادت‌ها و تایمر فوکوس است.

---

## ❌ این نسخه شامل چه مواردی **نیست**؟

در نسخه‌های بعدی، قابلیت‌های زیر اضافه شده‌اند که در این نسخه وجود ندارند:

- 📊 **آمار و نمودارهای پیشرفت**
- 📅 **تقویم شمسی کامل و مشاهدهٔ تاریخچه**
- 🧾 **خروجی PDF از گزارش هفتگی**
- ✅ **اعتبارسنجی کامل ورودی‌ها و مدیریت خطا**
- 🌙 **همگام‌سازی تم با سیستم**
- 📱 **ناوبری پایین برای موبایل**
- 🧩 **ماژولار شدن کامل کد**
- و بسیاری بهبودهای دیگر
